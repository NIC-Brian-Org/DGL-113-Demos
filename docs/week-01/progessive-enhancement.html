Progressive Enhancement

This page implements progressive enhancement.

"Progressive Enhancement" is a pessimistic development technique that "expects the worst" but "prepares for the best". If the browser doesn't support JavaScript, the user just sees a simple message below asking them to print the page page for their records. If the browser does supports JavaScript, a little script adds a "Print" button to the page. Progressive enhancement is the opposite of graceful degradation.

View the page source for details. To really see the effect, you will have to enable/disable JavaScript and refresh the page.

Please print this page for your records.
